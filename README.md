Calculadora de Juros por Atraso — 2,5% ao dia

Este programa em C# (Console) calcula o valor dos juros de um pagamento atrasado com base em:

Valor original
Data de vencimento
Multa de 2,5% ao dia de atraso
Data atual (automática)

É um programa simples, direto e ideal para o processo seletivo.

Tecnologias Utilizadas:

C#
.NET Console App
